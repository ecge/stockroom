# Stockroom

Stockroom is a commerce operations dashboard for teams that manage inventory,
orders, margins, and replenishment work behind an online store.

## What it demonstrates

- Inventory dashboard with units, restock watch items, order value, and margin.
- Searchable product table with operational status filters.
- Product detail panel with reorder guidance and live order state.
- Commerce admin interface that avoids cloned storefront branding.
- Responsive layout for operators who need dense information without clutter.

## Stack

- React 19
- TypeScript
- Vite
- Lucide React
- Plain CSS with reusable dashboard primitives

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
